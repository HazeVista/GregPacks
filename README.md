# GregPacks
## A GTCEu Modern Backpacks Mod

### What does this mod do?
This mod adds three tiers of "OmniPacks" with configurable stats, upgrades, and abilities. The mod is built to service any GTCEu Modern modpack to provide several useful QoL and utility features to the gameplay experience including:

- item, fluid, and energy storage upgrades
- flight upgrade
- processing upgrades
- magnet upgrade
- maintenance upgrade
- the ability to use GT machine covers and IO like a singleblock
